# 🤖 AI PDF Chatbot

An AI-powered PDF Question Answering chatbot built using Streamlit, FAISS, Sentence Transformers, and Groq LLM.

## Features

- Upload PDF
- Semantic Search
- FAISS Vector Store
- Sentence Transformer Embeddings
- Groq LLM
- English & Hindi Support
- RAG Pipeline

## Tech Stack

- Python
- Streamlit
- LangChain
- FAISS
- Sentence Transformers
- Groq
